Bipartition analysis of a given Graph

Assertions:
  1. Since no directions are pointed out in the task description, the graph is assumed to be undirected per default.
  2. Summarzed in short, a graph is bipartite if its every edge connects 2 vertices of different colors. Following this logic, a graph must not be connected to be bipartite.
  3. If a graph is disconnected, it can only be bipartite if all its disjoint sets are bipartite.
